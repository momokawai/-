日本三冠系列赛

每年4月

皋月赏

别称日本二千坚尼锦标赛，赛程1800米。1939年首次举行，仅限3岁马参赛。它是三冠赛首战，考验马匹速度与爆发力，为后续比赛奠定基础。

距离2000m 地点：中山竞马场
最快纪录 1分58秒0，由标志名驹2013年创下
每年5月

东京优骏

3岁雌马和雄马均可参赛，赛程2400米 。1932年创办，是日本最具影响力赛事，冠军马衣会被永久收藏于JRA博物馆。

距离2400m 地点：东京竞马场
最快记录 2分21秒1，由大震撼2005年创下
每年10月

菊花赏

别称日本圣烈治锦标赛，赛程3000米。1938年首办，允许3岁雄雌马参赛，注重马匹耐力，是三冠王的最后考验

距离3000m 地点：京都竞马场
最快记录 3分01秒，由东宝狐狼2022创下
